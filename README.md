## Featured Projects

### IDEACT — Intelligent SDG-Oriented Hackathon Advisor   `Minor Project | UPES 2025–26`
A web-based AI platform that helps students select and develop SDG-aligned hackathon projects. Takes a problem statement, runs NLP-based analysis, maps it to relevant UN Sustainable Development Goals, and generates a structured execution roadmap — not just answers, but a full plan.

**What makes it different:** Most hackathon platforms just list problems. SDG portals give theory. Generic chatbots give filler. IDEACT connects all three into a decision-support system that outputs real project guidance with feasibility scoring.

**Architecture:** 3-layer system — React.js frontend → Python/Flask REST API → PostgreSQL + MongoDB data layer, with an NLP processing module doing tokenization, keyword extraction, domain detection, and multi-SDG classification.

`Python` `Flask` `React.js` `PostgreSQL` `MongoDB` `NLP` `SDG Classification` `REST APIs`
